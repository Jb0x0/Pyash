# Contributing to Pyash

Thanks for your interest in improving Pyash!

---

## Code Contributions

1. Fork the repository.
2. Create a branch for your changes.
3. Make your changes.
4. Test your changes.
5. Submit a pull request.

---

## Reporting Bugs

Before opening an issue, make sure you are using the latest version of Pyash.

Please include:
* Pyash version
* Operating system
* Steps to reproduce the issue
* Expected behavior
* Actual behavior

---

## Feature Requests

Feature suggestions are welcome! Please explain:
* What the feature does
* Why it would be useful
* Any examples of how it could work

---

## Code Style

Please:
* Use clear variable and function names.
* Keep code readable.
* Add comments where needed.
* Avoid unnecessary dependencies.
